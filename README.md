# Nora Academy Website

This project is a responsive **frontend website** for Nora Academy.  
It includes multiple pages (Home, About, Courses, Contact) with a consistent navigation bar, styled sections, and a functional contact form.

---

## 🚀 Features
- **Global Navbar**
  - Logo + academy name on the left
  - Navigation links in the center
  - Donate button on the right (rounded, outline heart icon)
- **Pages**
  - **Home**: Introduction to Nora Academy
  - **About**: Information about the academy
  - **Courses**: List of available courses
  - **Contact**: Contact details and form
- **Contact Section**
  - Clickable links for email, phone, and WhatsApp
  - Contact form with Name, Email, and Message fields
  - On submit: Opens the user’s email app with the message pre‑filled to `Freshtaabbasi100@gmail.com`
